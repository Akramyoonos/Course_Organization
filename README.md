# EduGlobal - Degree Search Platform

A modern, responsive platform for searching over 120,000+ degrees across 210 countries.

## Features
- **Degree Search Engine**: Filter by subject and location.
- **Scholarship Directory**: Featured awards and categories.
- **Study Guides**: Country and subject detailed guides.
- **Virtual Fairs**: Events integration.
- **Student Support**: Webinars and expert advice.

## Tech Stack
- React + Vite
- Vanilla CSS (Glassmorphism, CSS Variables)
- Framer Motion (Animations)
- React Router (Navigation)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
